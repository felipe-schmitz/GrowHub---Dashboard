# Pagina Conceito da TerraSync
TerraSync integra gestão, análise de campo e inteligência artificial numa única plataforma criada para o produtor rural brasileiro — do pequeno sítio à grande fazenda.

## GrowHub Project 1.1
Dashboard interativo com mapa georreferenciado que centraliza todos os dados da propriedade: solo, clima, culturas plantadas e histórico de produtividade.
